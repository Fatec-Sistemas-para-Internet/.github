# Fatec – Sistemas para Internet

Organização criada por [@HelloAkiraS](https://github.com/HelloAkiraS) para centralizar os exercícios e atividades de programação desenvolvidos ao longo do curso de [Sistemas para Internet](https://fatecrl.edu.br/cursos/sistemas-para-internet) da [Fatec Baixada Santista Rubens Lara](https://fatecrl.edu.br). O objetivo principal é manter o repositório pessoal organizado, reunindo aqui os pequenos projetos feitos em aula — desde estruturas de dados até desenvolvimento web e orientação a objetos.

Os repositórios refletem o progresso acadêmico ao longo das disciplinas do curso, com códigos produzidos individualmente ou em grupo conforme a demanda de cada matéria. Por se tratarem majoritariamente de atividades pontuais de sala de aula, os projetos são intencionalmente simples e focados no conteúdo estudado no momento.

Para saber mais sobre a instituição e o curso, acesse o [site oficial da Fatec Rubens Lara](https://fatecrl.edu.br) ou a [página do curso de Sistemas para Internet](https://fatecrl.edu.br/cursos/sistemas-para-internet).
